# CMDB Service

## Adding a New Entity

### Step 1: Add the Table to Oracle

```sql
CREATE TABLE CMDB.WIDGET (
    ID NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    NAME VARCHAR2(100) NOT NULL,
    DESCRIPTION VARCHAR2(200),
    VENDOR_ID NUMBER NOT NULL REFERENCES CMDB.VENDOR(ID),
    CREATED_DATE DATE
);
```

### Step 2: Scaffold the Model from the Database

Re-run the EF Core scaffold command to regenerate models and context:

```powershell
dotnet ef dbcontext scaffold `
  "User Id=cmdb;Password=Cmdb1234;Data Source=localhost:1521/FREEPDB1;" `
  Oracle.EntityFrameworkCore `
  --output-dir Models `
  --context CmdbContext `
  --context-dir Data `
  --force
```
